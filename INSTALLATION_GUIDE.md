## 🔧 Customization

### Change Colors

Edit `styles.css`:

```css
/* Primary color (blue) */
.article-category { background-color: #dbeafe; color: #1e40af; }

/* Change to green */
.article-category { background-color: #dcfce7; color: #166534; }
```

### Add Your Logo

Replace in all HTML files:

```html
<a href="index.html" class="logo">EverydayMoments</a>

<!-- With -->
<a href="index.html" class="logo">
    <img src="your-logo.png" alt="Your Site">
</a>
```

### Update Contact Form

Edit `contact.html` and modify the form action:

```html
<form class="contact-form" action="https://formspree.io/f/YOUR_ID" method="POST">
```

### Add Google Analytics

Add before `</head>` in all HTML files:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

## 🌐 Deployment Options

### Option 1: GitHub Pages (FREE)

1. Create GitHub repository
2. Push all files to `main` branch
3. Go to Settings → Pages
4. Select `main` branch

### Option 2: Netlify (FREE)

1. Create account at netlify.com
2. Drag & drop the `public` folder
3. Site live instantly with custom domain support

### Option 3: Vercel (FREE)

1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel`
3. Follow prompts - done!

### Option 4: Traditional Hosting

1. Upload all files via FTP
2. Ensure index.html is in root
3. Done!

## 🛠️ Troubleshooting

### Python script doesn't run?

Make sure Python 3 is installed:

```bash
python3 --version
```

### Pages not linking correctly?

Ensure all files are in the same directory (public/)

### Mobile menu not working?

Check that script.js is loaded:

```html
<script src="script.js"></script>
```

### Contact form not submitting?

The form uses JavaScript simulation. To connect to a real backend:

1. Use Formspree.io (easiest)
2. Use Google Forms
3. Create your own backend API
