## 🐛 Troubleshooting

### Generator script won't run?
**Check Python 3 is installed:**
```bash
python3 --version
```

Should show: `Python 3.x.x`

### Pages don't load images?
Images are loaded from Unsplash CDN. Need internet connection to see them.

### Contact form doesn't submit?
It's a demo - uses JavaScript simulation. To make it real:
- Use Formspree.io (easiest)
- Connect to your own backend
- Use Google Forms

### Mobile menu doesn't work?
Make sure `script.js` is in the same folder and loaded.
